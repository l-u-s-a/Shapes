##Shapes

Java application for random generating shapes.

- download in current repo with this line: `git clone https://github.com/urfious/Shapes`

- run with: 
      
      ```
      java -cp bin:lib/picture.jar hr.fer.oop.lab1.topic2.prob1.Main
      ```
